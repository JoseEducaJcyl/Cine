# Cine

Aplicación web simple para gestionar un cine.

## Funcionalidades principales

- Ver cartelera de películas
- Reservar butacas en las salas
- Comprar entradas online
- Panel de administración para gestionar películas

## Instrucciones rápidas

1. Descarga o clona el proyecto
2. Abre en IntelliJ y ejecuta `main.java`
